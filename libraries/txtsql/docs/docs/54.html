INSERT Syntax
txtSQL >= 3.0.0 Beta

INSERT INTO ( db.table | table )
          SET ( column1 = value, column2 = value... ) |
          ([column1, column2...]) VALUES (column1_value, column2_value...)

The main purpose of this function is to insert a new record into the table/database combination specified.

If a column is set to auto_increment, you do not need to provide a 'pseudo' value as a substitute; the column will be incremented either way. If a column is of type enum and the value being inserted does not exist as one of the enum values, the LAST value from the enum list is inserted.

NOTE- To have a complete look at the INSERT function, and any changes to its functionalities, see the insert() function
