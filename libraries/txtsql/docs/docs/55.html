DELETE syntax
txtSQL >= 3.0.0 Beta

DELETE FROM [ db.table | table ]
          WHERE WhereClause
          [ LIMIT LimitClause ]

The main use for this statement is to eliminate rows from the table/database combination that match a certain criteria (the WHERE clause). If no WHERE clause in specified, then ALL rows will be deleted. Only rows that fall into the LimitClause will be dropped.

NOTE- To have a complete look at the DELETE function, and any changes to its functionality, see the delete() function
