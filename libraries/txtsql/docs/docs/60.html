USE Syntax
txtSQL >= 3.0.0 Beta

USE database

This statement sets a database as the default one (only during runtime), so that later on in other queries, there is no need to specify a database.

NOTE- To have a complete look at the USE function, and any changes to its functionality, see the selectdb() function
